# How Claude Code is Changing the Future of Coding

**Source**: [Lenny's Podcast - Boris Cherny, Head of Claude Code](https://www.youtube.com/watch?v=We7BZVKbCVw)
**Date**: 2025
**Duration**: ~1h 28m
**Speakers**: Lenny Rachitsky (Host), Boris Cherny (Head of Claude Code, Anthropic)

## Summary

Boris Cherny discusses the paradigm shift in software engineering driven by AI coding agents. He reveals that 100% of his code is now written by Claude Code, with engineering productivity up 200%. The conversation covers Claude Code's origin story, product philosophy, safety mechanisms, and predictions for the future of the software engineering role.
