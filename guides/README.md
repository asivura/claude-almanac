# Guides

Tutorials, setup walkthroughs, workflow recipes, and integration how-tos for Claude Code.

## What belongs here

Content that:

- Teaches the reader how to accomplish something
- Follows a step-by-step structure
- Has a defined outcome the reader achieves

## What doesn't belong here

- Objective feature descriptions → `../features/` (reference)
- Personal stories about specific projects → `../case-studies/`

## Categories

- **Setup** — environment and tool configuration
- **Workflow Recipes** — repeatable patterns
- **Integrations** — combining tools
- **Team Workflows** — multi-person patterns

## Writing a new guide

See the [content taxonomy](../site-planning/content-taxonomy.md) for full guidelines on voice, structure, and frontmatter schema.

All guides require the following frontmatter fields: `title`, `description`, `type: guide`, `category`, `time`, `difficulty`, `prerequisites`, `outcome`.
