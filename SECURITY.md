# Security Policy

## Reporting a vulnerability

If you discover a security issue in claude-almanac, please report it privately.
Contact: **alexander@amberflows.com**

Please include:

- A description of the issue
- Steps to reproduce
- Affected versions/commits

We'll respond within 7 days with a status and expected resolution timeline.

## Supported versions

Only the `main` branch is actively maintained. Security fixes are applied to `main` and deployed to claude-almanac.sivura.com on merge.
